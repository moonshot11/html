How to dine

!dine @USER
Dine at another user's restaurant. If USER is not provided, you dine at the streamer's restaurant!

!feed @USER
Feed USER. Will not work unless USER is provided.

Once you've had your meal, it's time to pay...or to dine and dash!

How to pay and rate

!give @USER POINTS
For example, to give 1000 points to Moonshot117, use !give @Moonshot117 1000

!rate1 @USER
!rate2 @USER
!rate3 @USER
!rate4 @USER
!rate5 @USER
You must use one of these commands to rate a restaurant. NOTE: The command !rate does not exist.

How to dine and dash

!dash
Flee the restaurant without paying. How dare you!

How to see your stats!

!rating @USER
View USER's rating. If no user is provided, you view your own rating.

!dashcount @USER
See how many times USER has dined and dashed! If no user is provided, you view your own count.
